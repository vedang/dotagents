# mck-consult

McKinsey consultative business problem solving system.

## Introduction

Systematize the McKinsey Problem Solving 101/102 methodology into an 8-step workflow, enabling end-to-end solutions from business problems to McKinsey-style PPTs.

## Core Abilities

- **Phase 1: Hypotheses Tree** - Problem Definition + MECE Disassembly + Hypothesis Driven
- **Phase 2: Dummy Pages** - Argumentative Design + McKinsey Page Layout
- **Phase 3: Data & Generation** - Intelligent Data Collection + Professional PPT Generation + Iterative Optimization

## Quick Start

```
"Please use mck-consult skill,
Help me analyze the growth and opportunities of the XX market in China "
```
