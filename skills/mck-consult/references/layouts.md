# McKinsey page layout library

7 classic page layout types and usage scenarios of McKinsey PPT.

## 1. Title + single chart type

**Structure**:
- Top: Argument-style title
- Middle: main chart (occupies 60-70% space)
- Bottom: Data Sources + Key Insights Box

**Applicable**: Single data story, trend display
**Chart type**: Line chart/Bar chart/Area chart

---

## 2. Title + left and right columns

**Structure**:
- Top: Argument-style title
- Left: Chart or bullet point list
- Right side: explanatory text or second diagram

**Applicable**: Comparative analysis, one picture and one article
**Common combinations**: chart + case text, key points + data table

---

## 3. Title + 2×2 matrix type

**Structure**:
- Top: Argument-style title
- Middle: 2×2 matrix graph (such as Boston matrix)
- Each quadrant: short descriptive text

**Applicable**: strategic positioning, classification framework
**Example**: Competition intensity × profit potential, growth rate × market share

---

## 4. Title + table type

**Structure**:
- Top: Argument-style title
- Middle part: multi-dimensional comparison table
- Header: dark blue background + white text

**Applicable**: Multi-dimensional comparison, scale analysis
**Example**: Competitive product comparison table, platform function comparison

---

## 5. Title + waterfall graph

**Structure**:
- Top: Argument-style title
- Middle part: Waterfall chart (growth breakdown/change attribution)

**Applies**: Breakdown of Growth Drivers
**Example**: Contributions of revenue growth from X to Y

---

## 6. Title + timeline type

**Structure**:
- Top: Argument-style title
- Upper part: Horizontal timeline
- Lower part: Bar chart/area chart for each time period

**Applicable**: Historical evolution, stage division
**Example**: Three stages of market development, technology evolution path

---

## 7. Insight summary page

**Structure**:
- Top: Summary title
- Middle part: 3-5 numbered insight boxes
- Each box: dark blue background + white title + light content area

**Applies**: Chapter summary, key findings
**Example**: Core insights and action recommendations

---

## Layout selection decision tree

```
Need to show trends? → Title + single chart shape (line/column)
Need comparative analysis? → Title + left and right column type or title + table type
Need strategic positioning? → Title + 2×2 Matrix
Need to break down growth? → Title + Waterfall Chart
Need historical evolution? → Title + timeline type
Need an insight summary? → Insight summary page
```
